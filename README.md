# Student Course Dropout Risk Prediction

## Overview

An end-to-end machine learning pipeline that predicts student dropout risk in university settings. Early identification of at-risk students helps academic administrators intervene before a student falls through the cracks.

---

## Pipeline Execution Order

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Generate synthetic dataset
python data/generate_data.py

# 3. Run ETL pipeline (clean & load into SQLite)
python database/etl_pipeline.py

# 4. Train all models, save best + metrics
python models/train_model.py

# 5. Launch the dashboard
streamlit run dashboard/app.py
```

## Models Trained

| Model               | Notes                              |
|---------------------|------------------------------------|
| Logistic Regression | Baseline linear classifier         |
| Random Forest       | Ensemble of decision trees         |
| Decision Tree       | Interpretable tree model           |
| Support Vector (SVM)| RBF kernel with probability output |
| K-Nearest Neighbors | Distance-based classifier          |

All models are wrapped in a `StandardScaler → Classifier` pipeline to ensure fair comparison.

---

## Evaluation Metrics

Each model is evaluated on a stratified 20 % hold-out test set using:

- **Accuracy** – overall correctness
- **Precision** – of flagged at-risk students, how many truly are
- **Recall** – of all at-risk students, how many are caught
- **F1-Score** – harmonic mean of precision and recall
- **Classification Report** – per-class breakdown (Not At Risk / At Risk)

## Features

| Feature                | Range   | Description               |
|------------------------|---------|---------------------------|
| `attendance_percentage`| 40–100  | % of classes attended     |
| `assignment_score`     | 35–100  | Assignment marks           |
| `midsem_score`         | 30–100  | Mid-semester exam marks   |
| `semester`             | 1–8     | Current academic semester |
| `course_difficulty`    | 1–5     | Perceived course difficulty|

**Label** (`dropout_risk`): `1` = At Risk, `0` = Not At Risk  
A student is At Risk when ≥ 2 of: attendance < 65 %, assignment < 50, midsem < 45, difficulty ≥ 4.
